(UI/UX Project) Clothing Webpage

Context

UI/UX project for a Clothing webpage

Role

Designed the UI, Conducted user research, Wireframing, Prototyping and Usability testing.

Timeline

Project completed in 4 weeks, including research, design, and testing.

Sitemap

Creating the storyboard.

Creating the wireframe

Implementation of the designs

Reflections

Project Achievements

I was able to successfully create a functional clothing webstore by incorporating key features such as login/registration, a product catalog, shopping cart, checkout process, and a feedback page. Additionally, I developed an administrator page that allows administrators to add new products to the catalog. By connecting the webstore with an SQL database, I ensured that both product and customer information are accurately reflected on the webpage and in the database.

Take Aways

However, due to time constraints and limitations in available tools, I was unable to implement features such as filters for different product brands across the catalog pages (tops and bottoms), as well as a payment details form in the checkout process, which was initially part of my wireframe design.Despite these challenges, this project provided valuable learning opportunities, particularly in coding techniques and problem-solving. It was an eye-opening experience, and I am proud of what I was able to achieve within the given constraints.
